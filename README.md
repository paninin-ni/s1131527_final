# 視窗（二）個人記帳與計畫管理系統

### 專案簡介:
> 提供直觀且具實用性的介面供使用者進行每日財務管理與長期的固定計畫收支預估 。系統支援「每日流水帳新增」，包含完整的「多條件歷史明細篩選」、「即時支出分類圓餅圖渲染」及「每月定期定額計畫交易自動融合」等核心機制。具備完善的防呆與驗證機制 ，能攔截非數字、小於或等於 0 的不合理金額輸入 並彈出警告提示。每次帳目變更皆會精確計算本月總收入、總支出與目前帳戶餘額 ，並擁有完整的檔案讀寫與外部匯入/匯出備份功能 ，讓不熟悉電腦操作的使用者也能流暢掌握個人財務狀態 。

### 快速記帳頁面（每日消費即時登錄）:
> 1. 本月財務儀表板
> * 本月總收入：自動統計並顯示當月累積的所有收入總和
> * 本月總支出：自動統計並顯示當月累積的所有支出總和
> * 目前帳戶餘額：即時計算淨資產（總收入 - 總支出）。餘額大於或等於 0 時顯示為黑色，若發生透支則自動變為紅色警示
> 2. 快速記帳表單操作
> * 日期選擇：點擊選單彈出日曆，選取該筆帳目的實際發生日期
> * 分類選取：提供「食、衣、住、行、育、樂、收入」下拉式選單，精確歸類帳目屬性
> * 金額輸入：於欄位內填入整數。系統具備防呆機制，若留白或輸入非數字、負數，將攔截並跳出錯誤提示
> * 備註填寫：可自訂輸入該筆款項的具體用途說明
> * 新增帳目：點擊「新增記帳」按鈕，系統會將資料寫入清單，更新儀表板，並在右側表格即時同步顯示最新 10 筆紀錄

### 歷史明細頁面（資料篩選與檔案管理）:
> 1. 歷史資料條件篩選
> * 時間區間：可自由調整「開始日期」與「結束日期」，鎖定特定範圍
> * 分類搜尋：可指定特定分類（如：只看「食」的支出）或選擇「全部」檢視
> * 關鍵字搜尋：於搜尋框內輸入文字，系統將自動比對備註欄中包含該關鍵字的帳目
> * 篩選與重設：點擊「篩選」更新下方表格；點擊「重設」則清空條件並回復完整清單
> 2. 帳目維護（即時刪除): 點擊歷史表格中任一資料列最右側的「❌ 刪除」按鈕，系統會跳出確認視窗，按下同意後即可將該筆紀錄自畫面與資料庫中徹底移除
> 3. 檔案備份與管理 (File I/O)
> * 匯出資料備份：點擊後調用儲存檔案視窗，可將所有記帳紀錄以 CSV 格式匯出至指定路徑，確保資料安全
> * 匯入歷史帳目：點擊後可載入先前備份的文字檔，一鍵還原所有歷史財務數據

### 統計計畫頁面（每月定期固定交易管理）:
> 1. 本月支出分析圓餅圖
> * 系統自動彙整「當月真實支出」與「當月生效的固定計畫支出」，即時渲染成圓餅圖
> * 圖表採用精緻的外側指引線模式，自動計算並清晰顯示如「食 45.0%」、「住 30.0%」等百分比標籤，消費結構一目了然
> 2. 新增每月固定計畫交易
> * 類型與分類：設定為「固定收入」（如家教薪水）或「固定支出」（如房屋租金、軟體訂閱）
> * 扣款日期設定：透過數字微調控制項（NumericUpDown）指定每月的第 $1 \sim 31$ 日為自動結算日
> * 有效期限控管：必須設定計畫的「開始月份」與「結束月份」，系統會自動進行日期合理性驗證（開始不可晚於結束）
> 3. 每月固定計畫交易清單 (dgvPlan): 點擊新增後，計畫會記錄於右側清單中。系統每逢重新整理，皆會聰明地比對當前月份是否處於計畫的有效期限之內。若條件吻合，該筆固定計畫金額將自動融入並加總至首頁的當月總收入與支出中
